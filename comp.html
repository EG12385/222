Below describes initial configuration of your company information



The following steps are performed only once after system installation:


Provide company information

1. On the Company info panel the following can be specified:

  • Company Name
  • Base Currency - Currently ZAR
  • Time Zone - Currently Africa/Johannesburg
  • ReRconcilliation Period

2. Address information is concerned with the legal information, the following can be updated:

  • "From" email for outgoing communication send from your billing environment
  • Contact info for invoices
  • Contact email

3. Use Legal adrress for invoices

The following information is required for invoicing and taxation:

  • Invoice number sequence
    • Individual for environment
    • Individual for reseller
    • Individual for customer


The following steps can be performed more than once, if and as needed:


Access Control

1. Left on the navigation menu, select My comapny, then Access control, and click Users.


2. Create user panel, is used to fill in the following user details:

  • Login
  • Email
  • Password
  • Role

3. Once all the user information was updated, select Save.



Roll

1. The following default roles are available:

  • Root
  • Admin
  • Helpdesk
  • No Access

2. Create Roll allows you to create a custom user roll by adding or removing parameters.


3. The following type of rolls can be defined:

  • Administrator self-care
  • Account self-care
  • Customer self-careS


Templates

Templates automate the processing of user input, wherby output can be customized in various media and data formats.

Through means of the built-in editor or by use of .html + .css, resellers can create their own invoice templates and customize with their brand logos, colors, layouts etc.


Adding a New Template, the following are available:

  • Source
    • Upload External Template
    • Design Template using Built-In Editor
  • Pages
    • A single page with totals only
    • First page with totals + usage details attached on additional pages
