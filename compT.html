Templates

Templates automate the processing of user input, wherby output can be customized in various media and data formats.

Through means of the built-in editor or by use of .html + .css, resellers can create their own invoice templates and customize with their brand logos, colors, layouts etc.



Adding a template

Adding a New Template, the following are available:

  • Name - Name description of the invoice
  • Type - By default set to 'Invoice'
  • Managed by
    • Administrato Only -By default
    • Invoice can be assigned to a reseller so that it is available in the list of templates on the reseller self-care interface
  • Source
    • Upload External Template - If you want to use an invoice template created offline
    • Design Template using Built-In Editor - If you want to create template using in-built functionality
  • Pages
    • A single page with totals only
    • First page with totals + usage details attached on additional pages

For Upload using External Template:

  • Template File - This allows you to upload a designed invoice template from an .html / .css file, simplifying the process and allowing advance template customization.
  • Add Image - You can add an image to the invoice template such as your company logo

To add your template file and images, on the left panel select Body and select your file and image to be used for the template. Take note the supported file formats for each field.



Once your files was upoaded you can selct the 'Preview' option to view the template.


Using the Built-In Editor:

Once the information was updated, on the top, select 'Launch Layout Designer'.

On the Built-In designer template, the infomation can be changed, removed and an image can also be added.
