Below describes the service catalog functionalities



The following steps are recommended to be done in order as listed below:


1. Rate Import

2. Bundles

3. Products


Rate Import

When operating as a VoIP service provider, you will want to be able to upload rate files and let the system process them automatically, in support of voice services rendered to your customers.

The following actions will be required:

  • Uploud rate data into a tariff
  • Start the import
  • Review file parsing
  • Review data field definition
  • Review rate information
  • Summary

Bundles

Volume Discount Plans

Volume discounts functionality allows you to adjust prices depending on the volume of services consumed by customers. Discount plan configuration provisions services that provides a certain number of "free" monthly minutes, or can provide discount to your customer after a certain number of calls have been made.

The following actions will be required:

  • Define a destination group
  • Create a volume discount plan
  • Define discounts within a volume discount plan
    • Create a discount
    • Define threshold
  • Assign a volume discount plan
    • Define a default volume discount plan to a product
    • Define a specific volume discount plan to a customer
    • Define a specific volume discount plan to a reseller
  • Check current volume discount statistics for an account

Subscription Plans

A subscription plan defines the charges associated with a particular service.

The following actions will be required:

  • Creating a subscription plan
  • Manage Fees
  • Promotional periods

Voice Products

Accounts for accessing your SIP services will be issued for a specific product. Products are a powerful tool defining different ways to bill an account. Product definition is always realized through these steps: product definition, service definition and configuration and creation of a rating list.

Main Product

The following actions will be required:

  • Create Product, name, type, managed by, account roll
  • Include services
  • Service Configuration
  • Usage Charges
  • Charges
    • Overdraft Protection
    • Default discount plan
    • Subscription
  • Additional Information
